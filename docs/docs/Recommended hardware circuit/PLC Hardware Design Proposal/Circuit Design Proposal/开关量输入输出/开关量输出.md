## 开关量输出
(1) 晶体管输出

- 源型输出,电平信号采用光耦隔离，输出端用三极管驱动，有效输出时电平为高电平，推荐电路如下图所示，此电路COM0口需外接电源，输出电压略低于电源电压。

![Image: image_029](../../images/image_029.png)
