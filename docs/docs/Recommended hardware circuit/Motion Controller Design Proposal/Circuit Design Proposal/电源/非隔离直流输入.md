## 非隔离直流输入

​非隔离方案中直流24V转5V采用DC/DC降压电路，推荐电路见下图所示，推荐电路设计输出为5V/2A。电源芯片型号为TPS54340，最大支持42V输入3.5A输出，支持100KHz到2.5MHz的固定或可调节开关频率。
