# 关于本站点

## 📖 站点简介

My MkDocs Site 是一个使用 MkDocs 构建的演示站点，旨在展示 MkDocs 的强大功能和简单易用性。

## 🎯 建站目标

- 提供一个清晰、简洁的文档展示平台
- 演示如何使用 MkDocs 创建专业的文档网站
- 分享 MkDocs 的使用经验和最佳实践
- 为用户提供一个可以参考和学习的示例

## 🛠️ 技术栈

- **构建工具**: [MkDocs](https://www.mkdocs.org) - 一个快速、简单的静态站点生成器
- **文档格式**: Markdown - 一种轻量级标记语言
- **主题**: mkdocs - MkDocs 默认主题
- **扩展**: 
  - toc: 生成目录
  - admonition: 添加提示框
  - codehilite: 代码高亮
  - extra: 额外的 Markdown 功能
